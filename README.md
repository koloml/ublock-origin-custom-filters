# uBlock Origin Custom Filters

Small set of custom filters for annoying parts of the sites I'm often using. This list is only used & tested for 
uBlock Origin extension.

## How to add

You can subscribe to specific filters by right-clicking the needed list and clicking the "Subscribe to filter list" 
option in context menu.
